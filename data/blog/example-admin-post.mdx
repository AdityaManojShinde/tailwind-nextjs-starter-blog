# Welcome to the Admin Dashboard

This is an example blog post created to demonstrate the admin dashboard functionality. You can now:

- **Create** new blog posts
- **Edit** existing posts
- **Delete** posts you no longer need
- **Search** through your posts
- **Manage drafts** and published content
- **Add header images** to make your posts more visually appealing

## Features

The admin dashboard includes:

1. **Password Protection** - Secure access to your admin panel
2. **WYSIWYG Editor** - Easy markdown editing
3. **Metadata Management** - Control titles, tags, summaries, and dates
4. **Draft System** - Save posts as drafts before publishing
5. **Header Images** - Add beautiful images to your blog posts
6. **Tag Management** - Create and manage your blog tags

## Getting Started

1. Navigate to `/admin` on your site
2. Enter your admin password
3. Start creating and managing your blog posts!
4. Add header images using URLs from services like Unsplash
